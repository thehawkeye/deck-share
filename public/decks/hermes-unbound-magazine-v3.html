Part I — What's Going On
The Economics

The
attention
economy
is dying.

For 27 years the internet ran on distraction as a business model. Google invented ads because credit card fixed fees killed micropayments in 1997. HTTP code 402, "Payment Required", sat unused for a generation.

LLMs do not get distracted. StackOverflow views down 75% since GPT-4. Tech news traffic down 60%. The contract is broken.

Attention to Outcomes: this shift disrupts ad-led models. It opens a new lane. Agent-to-Agent affiliate commissions. Your agent recommends. My agent transacts. Commission flows.

Market Reclassification
SaaS — The Old Addressable Market $200B

Software licenses, seat subscriptions, perceived pricing. Competes with other software.

AaaS — The New Addressable Market $4–5.5T

Agencies, outsourcers, professional services. Competes with every outcome a human currently produces. Source: Coatue, via Jason Shuman, March 2026.

"I would be alarmed if an engineer earning $500K a year did not consume at least $250K worth of tokens." — Jensen Huang, GTC

Token costs falling 40x in years. Agents will be 10x cheaper to run by 2028. Intelligence becoming "too cheap to meter" is the precondition for the consumer agent economy.

Part II — How Is That Happening
Consumer Agent Anatomy

The
agent
that
knows
you.

A consumer agent holds what no brand can: your preferences, your history, your budget, your context. Accumulated over every interaction.

The competitive advantage is not how many users an agent has. It is how well the agent knows this user. Depth compounds. Breadth does not.

Pre-articulation: the agent anticipates what you want before you say it. The agent-human flywheel compounds. More interactions gives deeper context, which enables better pre-articulation, which builds trust, which drives more delegation, which generates more signal.

The Delegation Boundary
RoutineNon-routine
Low criticality Full agent autonomy Agent recommends, human confirms
High criticality Agent pre-selects, human approves Human decides, agent assists

Full agentic commerce arrives first in low-stakes, high-frequency, pattern-established purchases. The top-right quadrant may never become fully autonomous.

The supervision spectrum runs from cruise control to Waymo. Trust earns autonomy. Architecture determines how much trust is required. This is a strategic decision, not a technical one.

Part III — What Does That Mean
The Central Thesis

The moat
migrates
from
breadth
to depth.

Platform era: more users means more data means more lock-in. The agent economy flips this. The new competitive advantage is not how many users you have. It is how well your agent understands each one.

Platforms whose main value prop is solving supply aggregation or discovery friction will be challenged, if not fully disrupted, by Agentic AI.

The Three Moat-Failure Modes — The Ken Scenarios
1 — Discovery Moat Collapse

Agents skip the browse.

Instamart's ad revenue collapses because agents do not see shelf placement. Flipkart's Big Billion Day visits drop from 600 million to under 200 million. Makemytrip's aggregation value becomes a single API call.

2 — Aggregation Moat Collapse

Agents route around the platform.

Zomato loses to direct restaurant ordering via ONDC at 18 to 22% savings. Cred's exclusive pre-approved loans become commodity in one agent scan for a 0.1 percentage point rate improvement.

3 — Open-Architecture Survival

Meesho Vaani: built for the agent era.

Zero-commission model became the routing default. Twenty-two percent higher conversion than app. Sixty percent of Rapido rides agent-initiated by 2028. Architectural alignment with agent access means survival.

The diagnostic question: which failure mode applies to your business? Discovery collapse means build ADO architecture. Aggregation collapse means become the open protocol. If neither applies: the depth moat is your only path.

Part III — What Does That Mean
Case Study

Walmart
x OpenAI:
Depth
beats
breadth.

October 2025. Walmart joins OpenAI's Instant Checkout inside ChatGPT. The vision: consumer asks for a product, agent finds it, checkout completes without leaving the chat.

The result: conversion rates roughly one-third of Walmart.com. Daniel Danker, EVP of AI, called it "unsatisfying." Linear chat cannot support multi-item baskets, bundling, or accessory recommendations.

Walmart's response: deploy Sparky, its own agent, inside ChatGPT. Users log in directly. Carts sync. Checkout runs on Walmart's infrastructure. Sparky converts at 70% of Walmart.com's rate. Double Instant Checkout.

The finding is unambiguous: consumers will delegate discovery to an AI agent. They will not delegate the transaction. The platform that owns the customer relationship beats the platform that owns the AI interface.

Platform Resilience Lens
Agent-Resistant

Value built on friction.

Attention capture. Information asymmetry. Pure aggregators. These face the most pressure as agents bypass discovery layers.

Agent-Replaceable

Low depth. Commodity offering.

Any agent can arbitrage these away in a single API call. No accumulated context. No trust. No moat.

Agent-Enhanced

Depth becomes the moat.

Platforms with strong physical or intangible assets, brand, know-how, R&D, compounded through deep personalisation, emerge stronger. Not in spite of agents. Through them.

Alignment is a major unsolved problem. Principal-Aligned versus Influenced, sponsored, agents. This leads to three agent business models: Subscription, Outcome-based, or Sponsored. Who funds the agent determines whose interests it serves.

India: The Open Rails Model

India
evolves
differently.

India's consumer agent model is structurally open. UPI solved payments. ONDC opened supply. Edge computing enables vernacular. The rails any agent can access. The agent routing layer is the new contested territory.

China takes the governed-bounded path. India takes the open-protocol path. Both will produce powerful consumer agent ecosystems, but with different winners.

The agent economy in India will evolve differently versus China. Open versus Bounded agents is not just a technical distinction. It is a regulatory and strategic one.

Part IV — What Should You Do About It
The Infrastructure Layer

The
protocols
that
matter.

Agentic commerce requires three layers to converge simultaneously: payment rails, compositional intelligence, and discovery infrastructure. Two implementations have emerged. The parallel to the 1990s is exact.

Google's UCP, January 2026: 20 plus partners including Walmart, Shopify, Flipkart, Target, Visa, Stripe. Open-source standard covering discovery, evaluation, transaction, settlement. The TCP/IP of commerce. Not a walled garden.

The Emerging Protocol Stack
MCP
Model Context Protocol. Context propagation between agents.
A2A
Agent-to-Agent communication. Coordination without human mediation.
AP2 / x402
Payment rails. Sub-cent transactions via stablecoins. Solved the fixed-fee problem that killed 1997 micropayments.
UCP
Universal Commerce Protocol. Discovery, evaluation, transaction, settlement. Google's open standard.
ADO
Agent Discoverability Optimization. The new SEO, but for APIs. Ranking factors: discoverability, reliability, capability, performance.

The practical result: an agent managing a household budget notices a 15% tariff-driven price increase, discovers three local alternatives, requests samples, negotiates volume pricing, and switches. All before the owner's morning workout. No procurement team.

Regulation must get ahead: KYA, Know Your Agent, norms, traceability, guardrails enforcement. India's KYA framework is not yet defined.
