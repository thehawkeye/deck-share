The Solution

Lock it
down to

Built on

The tech
stack behind
the magic.

Built with Next.js 14 App Router and TypeScript. Deployed on Vercel. Auth via HttpOnly cookies. Storage via Upstash Redis.

Magic links generated as UUID tokens. Tokens are single-use and expire in 24 hours by default.

Frontend

Next.js 14 App Router. CSS Modules. No Tailwind. Dark editorial theme. Fully responsive.

Auth

HttpOnly + Secure + SameSite=Strict cookies. Admin token in env vars. Per-recipient UUID tokens.

Storage

Upstash Redis for manifests and token state. Zero database setup. Scales to any number of decks.

Email

Resend for magic link delivery. One API call to send. HTML email with branded CTA button.
